# ITCHY YouTube Downloader v2.0 Guncelleme Notlari

Tarih: 27 Mayis 2026

Bugun yapilan guncellemeler:

- Program modern Electron + React masaustu arayuzune tasindi.
- Python indirme/analiz sistemi `python_core/worker.py` icine ayrildi.
- Windows icin modern installer paketi uretilecek sekilde `electron-builder` yapisi kuruldu.
- Installerin ileri adiminda kapanma ve kurulum sonrasi bos ekran acilma sorunlari giderildi.
- Uygulama surumu v2.0 olarak guncellendi.
- Yeni ITCHY Downloader logosu programa eklendi.
- Logo gorselinin icindeki bos alan kirpildi ve sol marka alanina daha buyuk yerlestirildi.
- Programin renk paleti logodaki yesil marka rengine gore yeniden ayarlandi.
- Dark/light tema destegi duzeltildi ve modern tema butonu eklendi.
- Ustteki `YouTube Media Studio` ve `Modern indirme paneli` metinleri kaldirildi.
- Sol menudeki `Indir`, `Kuyruk`, `Gecmis`, `Istatistik`, `Ayarlar` sekmeleri calisir hale getirildi.
- Kuyruk sayfasi eklendi ve link ekleme/kuyruk baslatma arayuzu hazirlandi.
- Gecmis sayfasi eklendi.
- Istatistik sayfasi eklendi ve worker tarafindaki gercek istatistik alanlariyla uyumlu hale getirildi.
- Ayarlar sayfasi eklendi.
- Native, koseli dropdownlar kaldirildi.
- Kalite ve altyazi dili secimleri icin modern, yuvarlak kenarli custom dropdown sistemi eklendi.
- Dropdownlarin kartlarin arkasinda kalma sorunu portal katmani ile giderildi.
- Format degistirildiginde kalite listesinin eski formattan kalma sorunu duzeltildi.
- Video linki analizinden sonra thumbnail gorunmeme sorunu icin yedekli thumbnail sistemi eklendi.
- Thumbnail yuklenemezse `maxresdefault` ve `hqdefault` kaynaklarina otomatik dusme eklendi.
- Program log sistemiyle paketli uygulamanin bos ekran acip acmadigi kontrol edilebilir hale getirildi.
- README dosyasi v2.0 surumune gore guncellendi ve logo eklendi.
